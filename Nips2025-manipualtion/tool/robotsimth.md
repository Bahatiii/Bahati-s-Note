# robotsimth
https://ar5iv.labs.arxiv.org/html/2506.14763

## 解决了什么问题：
机器人难以完成需要特定工具才能解决的复杂操作任务，现有的工具通常是为人类设计的，机器人抓取难以有效使用。而现有的方法局限于1工具对于机器人gripper不理想2受限于预定义模版3缺乏物理知识现有的通用3D生成方法无法对工具在特定任务的物理交互进行优化

## 论文提出了：
通过结合VLMs的物理知识和物理仿真的反馈，构建一个自动化pipeline：给定一个指定的操作任务，RobotSmith 会生成定制的工具几何形状，确定其在场景中的位置，并为机器人合成相应的操作轨迹，从而让机器人能够完成原本无法解决的复杂操作任务

由三个核心组件构成：
1. 评论工具设计器：两个协作VLM代理进行迭代，生成基于视觉上下文并以json格式描述的候选工具几何形状
2. 工具使用规划期：基于设计好的工作和场景集成操作轨迹
3. 联合优化器：微调工具几何形状和轨迹参数

## 独特性：
将工具设计、工具使用轨迹生成和任务性能优化整合到了一个端到端的生成和优化框架中，实现了协作VLM、工具几何和使用轨迹的联合优化、生成式解决工具使用场景的创新

## 如何解决：
1. critic tool designer 提议代理和批评代理迭代循环
2. tool use planner 生成工具的初始轨迹，表示为 三个最小API grasp release move的调用
3. optimizer (最关键的一步)
联合两个参数集：工具几何参数和工具使用轨迹参数，优化目标是最大化任务成功率和性能，利用仿真环境提供的精确物理反馈来指导优化
