# key word
泛化能力spatial generalization：在机器学习中，应用到不同场景的能力

鲁棒性robustness：在面对干扰仍能保持稳定可靠执行任务的能力

End-to-end:输入直接进模型 输出直接从模型出来，中间不需要人工设计的规则或子模块，能自动学习最优特征，模块间不会因为中间误差累计

Transformer:用self-attention来理解序列/图像/语言/动作的网络结构，有全局注意力，能对指令提及的逻辑关系自动联系，以此看到全部信息

扩散模型Diffusion:把目标加噪声-学习如何去噪声-最后从纯噪声一步步“去噪-生成合理动作”
π₀（pi-zero） 和 π₀.₅（pi-zero-point-five）：不使用某个模态、某个模块的策略，可能表示没有视觉、力觉、MOE的策略或者完全用原始VLA；部分使用某个模态、模块，只用一半权重的力觉

OpenVLA: 端到端 的通用机器人基础模型（robot foundation model），能把视觉（V）+ 自然语言（L） → 动作（A） 直接映射出来，用来控制机器人执行真实任务。示例输入：RGB图像或多视角图像、语言指令 输出：一个动作序列，例如位置变化、手抓夹紧度变化、机器人 末端执行器的速度、姿态变化。

视觉编码器-语言编码器-融合模态-transformer decoder-动作序列

Token:模型处理的最小信息单位，最小语义片段，图片也会被分成token，动作也可以是token，可能是六轴坐标，gripper开闭，末端速度方向，或连续动作用向量表示

对齐：观测和动作对应关系正确 一对一 看到这个应该做那个

MLP：多层感知机，输入层-多个隐藏层-输出层，每一层都包含多个神经元

### 读论文重点：
1. 解决了什么问题
2. 为什么能解决这个问题，有什么独特性，有什么平替
3. 数据如何对应研究问题（能不能想到其他更合适的数据）
4. 研究意义是什么（能不能想到更有高度的说法）
